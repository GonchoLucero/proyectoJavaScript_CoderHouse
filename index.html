Simulador De Central Eléctrica

Este simulador permite conectar electrodomésticos a la corriente eléctrica. Cada electrodoméstico tiene un consumo asignado y si este consumo supera el límite de seguridad maximo que tiene la corriente eléctrica, se activa la térmica y se corta la energía. El simulador tiene un menu principal el cual primero puedes observar los electrodomésticos y si se encuentran encendidos o apagados. Luego tiene la opcion de conectar los electrodomésticos, el cual si el consumo esta dentro de los parametros permitidos se enciende y sino salta la termica y el electrodoméstico no se enciende. Y por ultimo, tiene la opcion de salir del simulador.
